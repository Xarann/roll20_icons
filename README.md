# 🇬🇧 English Version

## 🎮 Roll20 HUD

> Custom Roll20 HUD for D&D 5e — fast, compact and immersive interface designed for real gameplay on the map.

---

## 🧠 Why this project?

Roll20 is powerful, but its default interface is not optimized for fast gameplay.
This HUD brings **all essential actions directly onto the map**, reducing clicks and improving flow during sessions.

---

## ✨ Features

* ⚡ Compact HUD centered on screen
* 🎯 Quick access to:

  * Attributes
  * Saving Throws
  * Skills (18 buttons)
  * Combat actions
* ❤️ HP Management

  * Current / Max / Temp
  * Hit Dice (DV)
  * Quick +/- controls
* 🎲 Roll modes

  * Normal / Advantage / Disadvantage
  * Synced with character sheet
* 🧪 Dynamic Resources

  * Auto-detection
  * Increment / decrement
* 🛡️ Global Modifiers

  * Attack / Damage / Saves / AC
* ⚔️ Combat system

  * Auto-detects weapons
  * One-click actions
* 🎨 Clean dark fantasy UI

## 🚀 Installation

## 🧩 How it works

The HUD is built around 3 main areas:

* **Main bar** → stats & roll modes
* **Action buttons** → navigation
* **Dynamic panel** → contextual content

---

## ⚠️ Limitations

* Depends on Roll20 DOM (may break with updates)
* Designed for D&D 5e sheets
* Requires Tampermonkey
